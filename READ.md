# 🤖 AI 批量图像去水印工具 — 使用手册

[cite_start]感谢您使用本工具！本程序基于最新的 **LaMa AI 模型**，专门针对高清图片（如 2730×1535 分辨率）的水印移除进行了深度优化 [cite: 1]。

---

## 📥 第一步：环境准备（仅需执行一次）

[cite_start]在使用本工具前，您的电脑需要具备 **Python** 运行环境 [cite: 1]：

1. **安装 Python**:
    * [cite_start]请前往 [python.org](https://www.python.org/) 下载并安装最新版本 [cite: 1]。
    * [cite_start]⚠️ **重要**：安装时请务必勾选 **"Add Python to PATH"**（将 Python 添加到系统变量） [cite: 1]。
2. **配置 AI 引擎**:
    * [cite_start]找到文件夹中的 `首次使用请双击我.bat` [cite: 1]。
    * [cite_start]**双击运行**，程序会自动为您下载并配置 AI 模型及相关组件 [cite: 1]。
    * [cite_start]💡 **注意**：由于模型文件较大，建议在网络环境良好的情况下进行，耗时约 **3-5 分钟** [cite: 1]。

---

## 🚀 第二步：开始处理图片

### 1. 准备原图
* [cite_start]建议将需要处理的图片统一放入一个新建的文件夹中（例如：`D:\待处理图片`） [cite: 1]。
* [cite_start]🛡️ **安全提示**：请勿直接选择 C 盘根目录、Windows 文件夹或 Program Files 文件夹，程序会自动拦截这些敏感目录以保护系统 [cite: 1]。

### 2. 启动工具
* [cite_start]双击运行 **`去水印工具.exe`** [cite: 1]。

### 3. 操作步骤
1. [cite_start]点击界面上的 **“选择文件夹并开始处理”** 按钮 [cite: 1]。
2. [cite_start]在弹出的窗口中选中刚才建立的文件夹（如 `待处理图片`） [cite: 1]。
3. 点击确定后，程序会自动执行以下操作：
    * [cite_start]**创建 masks 临时文件夹**：用于存储遮罩（处理完后会自动删除） [cite: 1]。
    * [cite_start]**创建 result 文件夹**：用于存放处理后的无水印成品 [cite: 1]。

### 4. 查看结果
* [cite_start]**进度显示**：处理进度会在工具界面实时更新 [cite: 1]。
* [cite_start]**成品路径**：完成后，您可以直接进入原图路径下的 **`result`** 文件夹查看成品 [cite: 1]。

---

## 📝 注意事项与常见问题

* [cite_start]**处理速度**：由于工具处理的是高清大图，且默认使用 **CPU** 进行高精度运算，单张图片处理时间约为 **5-15 秒**（取决于电脑性能），请耐心等待 [cite: 1]。
* [cite_start]**水印位置**：本工具目前默认移除图片 **右下角** 区域的水印 [cite: 1]。
* [cite_start]**报错提示**：如果弹出“未检测到 Python”或“安装失败”，请检查网络连接，并确保安装 Python 时勾选了 **PATH** 选项 [cite: 1]。
* [cite_start]**系统拦截**：如果您选择的路径提示“敏感目录”，请将图片移动到非系统盘（如 D 盘或 E 盘）再次尝试 [cite: 1]。
